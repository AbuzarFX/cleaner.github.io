Cleaner Bot for Discord

Cleaner is a bot that can delete a variety of different messages, whether it be messages that begin with the most common command symbols, messages by a certain user or messages sent by bots, there is a command/parameter that'll help clean up your chats regardless of what's posted.

Add To Your Server

Cleanup

Pairing this command with parameters gives you MAXIMUM chat moderation control requires permissions: MANAGE MESSAGES

Invite

Get an invite link to add the bot to more servers.

Help

Sends helpful documentation about commands, their usages, their aliases and even a short description.

Stats

Displays information about the bot and server performance. An embed with all information is displayed, however you can provide a valid shard ID to get specific details.

Parameters

Text parameters
  • before (message id | message link | timeframe)

    Targets messages posted before the specified message id, message link or timeframe.

    after (message id | message link | timeframe)

    Targets messages posted after the specified message id, message link or timeframe.
Image parameters
  • attachments aliases: files

    Deletes messages with file attachments (not images)

    Valid extensions are a fullstop followed by the extension, example: .mov images aliases: imgs Deletes messages with image attachments (not files) Valid image extensions are webp, png, jpg, jpeg & gif
